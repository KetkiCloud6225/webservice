# Web Server

## Prerequisites
    
1. Express JS
2. jest (for unit testing)
3. mySql
4. Sequelize
5. multer
6. aws-sdk
   
## Building the application

1. Clone the repository

    ```sh
    git clone git@github.com:KetkiCloud6225/webservice.git
    ```


2. Run the following command to compile the code
   
   ```
   node index.js
   ```
